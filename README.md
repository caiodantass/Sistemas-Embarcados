# 📡 Sistemas Embarcados

Repositório com **atividades de laboratório da disciplina de Sistemas Embarcados**, organizadas por exercício/atividade.  
Feito pelo aluno **Caio Lívio Leite Muniz Dantas** como parte dos estudos da matéria. :contentReference[oaicite:1]{index=1}

---

## 🧠 Descrição

Este projeto reúne uma coleção de exercícios e atividades práticas realizadas ao longo da disciplina de **Sistemas Embarcados**.  
O conteúdo foca principalmente em conceitos fundamentais da área de sistemas embarcados — como microcontroladores, periféricos, atuação direta com hardware, comunicação e programação de baixo nível — aplicados em linguagem C (ou outra linguagem recomendada no curso). :contentReference[oaicite:2]{index=2}

---

## 📂 Estrutura do Repositório

O repositório está organizado em pastas/arquivos de **atividades numeradas**, por exemplo:

Atividade 01 - Sistemas Embarcados
Atividade 02 - Sistemas Embarcados
…
Atividade 10 - Sistemas Embarcados



Cada pasta contém os arquivos específicos da atividade proposta pela disciplina. :contentReference[oaicite:3]{index=3}

---

## 📌 Como usar este repositório

Este repositório é didático e pedagógico, voltado para:

- 📍 Estudo de conceitos de sistemas embarcados
- 📍 Referência de soluções e exemplos das atividades
- 📍 Consulta rápida ao código produzido na disciplina

Para ver o código de alguma atividade, basta acessar a pasta correspondente e visualizar os arquivos.

- 📡 Conceitos de hardware e software integrados
